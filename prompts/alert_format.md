# 알람 출력 형식 가이드

> AI 분석 결과의 표준 출력 형식

---

## 심각도 레벨

| 아이콘 | 심각도 | 설명 | 조치 권장 |
|--------|--------|------|-----------|
| 🔴 | 높음 (High) | 즉시 확인 필요 | 반드시 확인 후 진행 |
| 🟠 | 높음 (High) | 범위 이탈 | 수치 재확인 필요 |
| 🟡 | 중간 (Medium) | 누락/미완료 | 보완 필요 |
| 🔵 | 낮음 (Low) | 형식 문제 | 수정 권장 |
| ⚪ | 확인필요 | 판단 어려움 | 담당자 검토 필요 |

---

## 문제 유형별 아이콘

```
🔴 값 불일치 (Value Mismatch)
   - 기준값과 측정값이 다름
   - 예: 기준 12.0V, 샘플 15.0V

🟠 범위 이탈 (Out of Range)
   - 허용 오차 초과
   - 예: 기준 12.0V ±0.5, 샘플 12.8V

🟡 누락 항목 (Missing Items)
   - 필수 항목 누락
   - 예: FT-015 항목 전체가 없음

🟡 미완료 항목 (Incomplete)
   - 필수 필드 빈 값
   - 예: 서명 누락, 날짜 미기입

🔵 형식 오류 (Format Error)
   - 데이터 형식 불일치
   - 예: 날짜가 YYYY/MM/DD 대신 MM-DD-YYYY

⚪ 의심 항목 (Suspicious)
   - 추가 확인 필요
   - 예: 연속 동일값, 비정상 패턴
```

## 요약 테이블 형식

```markdown
## 📊 분석 요약

| 유형 | 건수 | 심각도 |
|------|------|--------|
| 🔴 값 불일치 | 3건 | 높음 |
| 🟠 범위 이탈 | 2건 | 높음 |
| 🟡 누락 항목 | 1건 | 중간 |
| 🟡 미완료 항목 | 4건 | 중간 |
| 🔵 형식 오류 | 2건 | 낮음 |
| ⚪ 의심 항목 | 1건 | 확인필요 |

**총 13건의 확인 필요 항목 발견**
```

## 상세 항목 형식

### 값 불일치 항목

```markdown
### [1] FT-005 전압 테스트
- **위치**: Main Sheet > Row 7
- **기준값**: 12.0 V
- **샘플값**: 15.3 V
- **차이 설명**: 기준 대비 +27.5% 차이 (3.3V 초과)
```

### 범위 이탈 항목

```markdown
### [1] FT-008 전류 테스트
- **위치**: Main Sheet > Row 10
- **기준값**: 2.5 A
- **허용 범위**: 2.4 ~ 2.6 A (±0.1)
- **샘플값**: 2.8 A
- **이탈 정도**: 상한 대비 +0.2 A 초과
```

### 누락 항목 테이블

```markdown
| 항목번호 | 항목명 | 비고 |
|----------|--------|------|
| FT-015 | 절연 저항 테스트 | 전체 행 누락 |
| FT-016 | 내전압 테스트 | 전체 행 누락 |
```

## 코멘트 섹션 형식

```markdown
## 💡 분석자 코멘트

1. **가장 주의가 필요한 항목**:
   - FT-005 (전압 27.5% 초과) - 안전 관련 항목으로 반드시 재측정 권장

2. **패턴 발견**:
   - FT-020~FT-025 구간에서 동일한 값(0.00)이 반복됨
   - 측정 장비 오류 또는 입력 오류 가능성

3. **추가 확인 권장**:
   - 서명 누락 4건 - 담당자 확인 필요
   - FT-015, FT-016 누락 - 의도적 제외인지 확인
```
